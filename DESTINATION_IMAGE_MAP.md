# DESTINATION IMAGE MAP
## YouNew - Visual Identity Map for Dutch Cities and Places

Version: 1.0
Date: 2026-06-16
Owner: Travel Content Art Direction / Information Architecture / Asset Pipeline
Status: Canonical target map

---

## 1. Rule

Every destination must have a visual identity before an image can be approved.

Do not assign random "pretty Netherlands" images.
Do not reuse one canal, windmill, street, or skyline image across unrelated destinations.
Do not allow a generic fallback to become the visible production identity of a city.

---

## 2. Destination Image Slots

Every destination must have:

| Slot | Required | Purpose |
|---|---:|---|
| Primary hero image | Yes | Destination identity on detail/home/map surfaces |
| Secondary images | 2-4 | Topic, culture, POI, and local life variety |
| List thumbnail | Yes | Cities/provinces/search lists |
| Square search image | Yes | Search, saved, AI related cards |
| Fallback image | Yes | Only approved fallback; never random or generic |

## 4. Variant Requirements

For every destination above:

| Variant | Aspect Ratio | Minimum Output | Crop Rule |
|---|---:|---:|---|
| `hero` | 16:9 | 2400 x 1350 | Landmark visible in center 60%; safe under dark top/bottom overlay |
| `card` | 4:3 or 5:4 | 1600 short-side target | Main object not cropped by rounded card frame |
| `tallCard` | 4:5 | 1600 short-side target | Vertical composition keeps skyline/landmark readable |
| `squareSearch` | 1:1 | 1200 x 1200 | Identity remains readable at small size |
| `thumbnail` | 1:1 or 4:3 | 800-1200 px | No tiny text/detail as primary identifier |

---

## 5. Screen Assignment Rules

| Screen | Required Image Slot |
|---|---|
| Home city hero | destination primary hero |
| Cities directory tile | list thumbnail |
| City detail hero | destination primary hero |
| City attractions | POI-specific image, not city hero unless POI is the hero landmark |
| Province detail | province primary hero, not a duplicate city hero where avoidable |
| Map preview | list thumbnail or square search image |
| Search result | square search image |
| Saved card | same approved image ID as original item |
| AI related card | same approved image ID as recommended content |
| Culture & Attractions | theme-specific secondary image with city/place context |
