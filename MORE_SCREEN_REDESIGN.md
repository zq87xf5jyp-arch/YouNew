# More Screen Redesign

Date: 2026-06-14

## Before

The More screen had a useful Personal Guide Dashboard, but the lower Profile block still exposed:

- Privacy & Data Control
- Knowledge diagnostics
- Change Situation
- About the App

This made the screen feel administrative and partially internal.

## After

The main More path now keeps practical user-facing areas first:

- Current city
- Weather
- Emergency
- Saved items
- Documents
- Language learning
- Transport
- AI Assistant
- Recent activity
- Important deadlines

The lower section is now a minimal Account section with Settings only. Change Situation remains available as a top quick action because it changes personalization and belongs higher in the flow.
