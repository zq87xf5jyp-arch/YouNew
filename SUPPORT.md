# YouNew support

## Product help

Visit [younew.nl/support](https://younew.nl/support/) or email
[support@younew.nl](mailto:support@younew.nl) for ordinary product support.
Include the app version, iOS or iPadOS version, device model, language, and a
short description of what happened.

## GitHub issues

Use the repository issue forms for reproducible software bugs, feature requests,
and corrections to official-source content. Search existing issues first and
keep one problem per issue.

Never post a BSN, identity document, home address, medical information,
credential, private correspondence, or other personal data. Redact screenshots
before attaching them and confirm that you have the right to share every image.

## Security and emergencies

Report vulnerabilities privately according to [SECURITY.md](SECURITY.md). Do
not use a public issue for exploit details or secrets.

YouNew is not an emergency service. In the Netherlands, call `112` for an
immediate life-threatening emergency and verify non-emergency guidance through
the relevant official service.
